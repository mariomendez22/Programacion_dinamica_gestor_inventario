## Sistema de Inventario en C++

Programa en C++ que simula un sistema básico de inventario mediante un menú en consola, permitiendo administrar productos almacenados en un arreglo.
## Funciones principales

--Mostrar todos los productos

--Buscar productos por ID

--Buscar productos por nombre

--Ordenar productos por precio

--Ordenar productos por cantidad

--Modificar la cantidad de un producto

## Notas

--El inventario se inicializa con 10 productos.

--El programa valida las entradas del usuario.

--Los errores se manejan mediante excepciones.

## Instrucciones de uso

Clonar o descargar el repositorio:
```bash
https://github.com/mariomendez22/Programacion_dinamica_gestor_inventario
```
Compilar el programa
```bash
g++ gestor_inventario.cpp -o inventario
```
Ejecutar el programa
```bash
./inventario
```
